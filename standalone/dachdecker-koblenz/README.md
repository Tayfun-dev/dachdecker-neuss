# Dachdecker Koblenz Standalone

Dieser Ordner ist repo-fertig für eine eigene GitHub-Pages-Seite.

So bekommst du einen sauberen Link ohne `dachdecker-neuss`:

1. Neues GitHub-Repo anlegen:
   `dachdecker-koblenz`
2. Den kompletten Inhalt dieses Ordners in das neue Repo hochladen.
3. In GitHub `Settings` -> `Pages` -> `GitHub Actions` auswählen.

Dann ist die Seite unter dieser URL erreichbar:

`https://tayfun-dev.github.io/dachdecker-koblenz/`

Dateien in diesem Ordner:

- `index.html`
- `styles.css`
- `.github/workflows/pages.yml`
